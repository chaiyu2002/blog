Git的git-checkout的用法总结。

初衷

git-checkout是Git最常用的命令之一，但又是有些复杂的命令，总会感觉有些用不明白，用不明白的原因应该是没有深度地、全面地理解一下，所以要对它好好整理一下。

介绍

checkout在CVS和SVN中都是检出的意思，从版本库检出一个版本，在Git中就不是这么简单了。手册上是这样介绍的：

在Git里面，checkout用于切换分支或者恢复工作树的文件。
